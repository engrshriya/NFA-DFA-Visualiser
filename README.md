# NFA to DFA Visualizer

This project demonstrates the subset construction method used to convert a Non-Deterministic Finite Automaton (NFA) into a Deterministic Finite Automaton (DFA).

Features:
- User input of NFA
- Subset construction algorithm
- DFA transition table generation
- Graphical visualization of NFA
- Graphical visualization of DFA

Technologies Used:
- HTML
- CSS
- JavaScript
- Vis Network Graph Library

How to Run:
1. Download the project
2. Open index.html in a browser
3. Enter NFA details
4. Click "Convert NFA → DFA"
